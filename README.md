# سجلّ الأرباح — دفتر الصفقات

تطبيق React مستقل (خارج بيئة Claude) لتتبع صفقات الأسهم مع تحديث سعر تلقائي عبر Finnhub.

## 1) تشغيله محليًا (اختياري، للتجربة قبل النشر)

```bash
npm install
npm run dev
```

يفتح على `http://localhost:5173`

## 2) نشره على Vercel (أسهل طريقة — بدون سطر أوامر)

1. روح إلى https://vercel.com وسجّل دخول (تقدر تسجل بحساب GitHub مباشرة)
2. اضغط **Add New… → Project**
3. اختر **"Upload"** أو اسحب مجلد `trading-ledger-app` كامل (لو ما فيه خيار رفع مباشر، ارفعه أول على GitHub — خطوات تحت)
4. Vercel بيكتشف تلقائيًا إنه مشروع Vite. اضغط **Deploy**
5. بعد دقيقة رح تحصل على رابط مباشر مثل `trading-ledger-xxxx.vercel.app` — هذا رابط تطبيقك الحي

## 2ب) طريقة GitHub (الأنسب للتحديثات المستقبلية)

1. أنشئ مستودع جديد فاضي على https://github.com/new
2. من داخل مجلد المشروع:
   ```bash
   git init
   git add .
   git commit -m "أول نسخة"
   git branch -M main
   git remote add origin <رابط-المستودع>
   git push -u origin main
   ```
3. بموقع Vercel: **Add New… → Project → Import Git Repository** واختر المستودع
4. اضغط **Deploy**

كل مرة تعمل `git push` بعدها، Vercel ينشر النسخة الجديدة تلقائيًا.

## 3) بعد النشر

- افتح الرابط، اضغط ⚙️ وحط مفتاح Finnhub الخاص بك
- فعّل "تحديث تلقائي" — بما إنه الآن تطبيق ويب حقيقي بدون قيود Sandbox، الاتصال بـ Finnhub رح يشتغل طبيعي
- البيانات تُحفظ بمتصفحك (localStorage) — يعني خاصة فيك على هذا الجهاز/المتصفح فقط

## ملاحظة أمان

مفتاح Finnhub يبقى مرئي بالمتصفح (كود Frontend). هذا مقبول لاستخدام شخصي، بس لا تشارك رابط التطبيق مع أشخاص ما تثق فيهم لأنه بيقدرون يشوفون المفتاح من أدوات المطوّر بالمتصفح.
